   - Şu an "TarımPazarı ile İşinizi Dijitalleştirin" yazısının arkası BEYAZ görünüyor. Bunu düzelt.

   - **Görsel URL:** `https://images.unsplash.com/photo-1625246333195-09d9b430db80?q=80&w=1920&auto=format&fit=crop` (Geniş, yeşil, modern bir tarım arazisi).

     ```css
     background: linear-gradient(rgba(0, 0, 0, 0.7), rgba(0, 0, 0, 0.7)), url('[https://images.unsplash.com/photo-1625246333195-09d9b430db80?q=80&w=1920&auto=format&fit=crop](https://images.unsplash.com/photo-1625246333195-09d9b430db80?q=80&w=1920&auto=format&fit=crop)');

     ```
